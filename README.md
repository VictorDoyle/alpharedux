## Run

Development server available on terminal by running:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) on browser.


## Styling
Font: Futura or Jost 
Font-weight: ['200','300', '500'],

### Color Palette

Dark mode:
Background: #141414
Font/headings/p : #ffffff

Light mode:
Background: #ffffff
Font/headings/p : #000000
